#KPLC Smart Token Manager
A comprehensive mobile application designed to streamline the electricity token purchasing experience for Kenya Power (KPLC) customers. This app eliminates the tedious process of manually entering 20-digit token codes by automatically updating your meter upon purchase.
#Key Features
##Automated Token Management

Purchase tokens directly through the app with automatic meter updates
No more manual entry of long token codes
Instant synchronization with your prepaid meter via a smart bridge device

##Real-Time Monitoring

Track your electricity consumption in real-time
View usage patterns and historical data
Get low balance alerts before running out of power

##Smart Auto-Renewal

Set custom threshold amounts for automatic top-ups
Seamless STK push integration for quick payments
Never experience unexpected power outages again

##Multi-Meter Support

Add and manage multiple meters from a single account
Perfect for landlords, property managers, or households with multiple meters
Easy switching between different meters
Individual monitoring and management for each meter

##Bridge Technology

Utilizes a hardware bridge to communicate with KPLC meters
Secure, encrypted communication
Compatible with standard KPLC prepaid meters
